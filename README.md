# ros2-diffdrive-robot

> A personal learning project — building a differential-drive robot from scratch using ROS2, starting software-first in simulation and eventually moving to real hardware.

I started this project out of curiosity and a genuine interest in robotics. As a master's student in Sports Technology & Engineering with a background in embedded systems and sensor fusion, I wanted to get hands-on with ROS2 and build something real — not just follow tutorials.

The goal is to go from zero ROS2 knowledge to a fully functional, self-built robot. Everything here is learned as I go, so expect iterative improvements and honest commit history.

---

## Project Roadmap

- [ ] **Phase 1 — ROS2 Basics**: nodes, topics, services, transforms (TF2)
- [ ] **Phase 2 — Simulation**: URDF model + Gazebo, differential drive controller
- [ ] **Phase 3 — Localization**: encoder odometry + IMU sensor fusion
- [ ] **Phase 4 — Navigation**: Nav2 stack, basic autonomous navigation
- [ ] **Phase 5 — Hardware**: Raspberry Pi + ESP32/Arduino, 3D-printed chassis
- [ ] **Phase 6 — Integration**: full system running on real robot

---

## Stack

- **ROS2** (Humble / Jazzy)
- **Gazebo** for simulation
- **Python & C++** for nodes
- **Raspberry Pi** (planned hardware target)
- **ESP32 / Arduino** for low-level motor control
- **IMU** for sensor fusion
